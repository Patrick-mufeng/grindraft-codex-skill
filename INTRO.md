# 磨稿 / Grindraft

> Claude Code 插件 · 18 个 skill · 把公众号写作变成可进化的校准系统

## 一句话

**不是帮你写文章，是帮你建立一套"每写一篇，下一篇就更准"的写作系统。**

## 和普通 AI 写作工具有什么区别？

普通 AI 写作写完就结束，不记得你上一篇阅读量。
磨稿把每一篇文章变成数据，数据反过来校准你的写作判断。
